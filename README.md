
# 🚀 Resume Analyzer

An intelligent web application that analyzes PDF resumes against job descriptions to provide a compatibility score, skill gaps, and improvement suggestions. Built with **Flask** and **Python**.

## ✨ Features
* **PDF Text Extraction**: Uses `PyPDF2` to read and process resume content.
* **Skill Matching**: Automatically detects technical skills like Python, Java, SQL, and more.
* **Job Description Comparison**: Calculates a match percentage based on the specific job role.
* **Smart Suggestions**: Provides actionable tips like adding GitHub links or project experience.
